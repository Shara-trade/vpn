# 💰 Процесс покупки тарифа

## Обзор

Покупка тарифа — это основная monetization функция бота. Пользователь выбирает тариф, подтверждает покупку, средства списываются с баланса и генерируется VPN ключ.

## 📝 Callback Data

| Callback | Описание |
|----------|----------|
| `buy_tariff_{id}` | Выбор тарифа (id = 1,2,3,4) |
| `confirm_purchase_{id}` | Подтверждение покупки |
| `cancel_purchase` | Отмена покупки |
| `back_to_main` | Возврат в меню |

## 🔧 Основные функции

### `show_tariffs(message, db_user)`

Показывает список тарифов с ценами и кнопками.

```python
await show_tariffs(message, db_user)
```

**Логика:**
1. Получает тарифы из БД
2. Формирует сообщение с ценами
3. Создаёт inline-клавиатуру
4. Помечает доступные/недоступные тарифы (✅/❌)

---

### `callback_buy_tariff(callback, db_user)`

Обрабатывает выбор тарифа.

**Логика:**
1. Получает тариф по ID
2. Проверяет баланс пользователя
3. Если не хватает — показывает ошибку
4. Если хватает — показывает подтверждение

**Пример ответа:**
```
✅ Ты выбрал: 3 месяца (799 ₽)

Твой баланс: 1 250 ₽
Баланс после покупки: 451 ₽
💰 Экономия: 100 ₽

Подтверждаешь покупку?
[✅ Подтвердить] [❌ Отмена]
```

---

### `callback_confirm_purchase(callback, db_user)`

Выполняет покупку.

**Шаги:**

1. **Валидация**
   - Проверяет тариф
   - Проверяет баланс (двойная проверка)
   - Проверяет наличие серверов

2. **Списание средств**
   ```python
   new_balance = await queries.update_user_balance(user_id, -price)
   ```

3. **Расчёт даты истечения**
   ```python
   if user.has_active_subscription:
       expires_at = current_expires + days
   else:
       expires_at = now + days
   ```

4. **Генерация ключа**
   ```python
   xui = XuiService(server)
   key_data = await xui.create_client(user_id, days)
   ```

5. **Сохранение**
   - Установка даты истечения
   - Сохранение ключа
   - Создание транзакции

6. **Ответ пользователю**
   - Показывает успех + ключ
   - Или ошибку с возвратом средств

## ⚠️ Обработка ошибок

### Недостаточно средств
```
❌ Недостаточно средств на балансе!

Тариф: 3 месяца (799 ₽)
Твой баланс: 150 ₽
Не хватает: 649 ₽

💬 Напиши @FreakVPN_Shop для пополнения баланса
```

### Нет серверов
```
❌ Нет доступных серверов. Обратитесь в поддержку.
```

### Ошибка генерации ключа
```
✅ Оплата прошла успешно!
...
⚠️ Не удалось автоматически создать ключ.
Напиши в поддержку @FreakVPN_Support для получения ключа.
```

### Критическая ошибка
```
❌ Произошла ошибка при обработке покупки.

💰 Средства возвращены на ваш баланс.
Попробуйте позже или обратитесь в поддержку.
```

## 🔒 Безопасность

1. **Двойная проверка баланса** — перед списанием и при подтверждении
2. **Возврат средств** — при любой ошибке средства возвращаются
3. **Логирование** — все операции логируются
4. **Транзакции БД** — атомарные операции

## 📊 Транзакции

Каждая покупка создаёт запись в таблице `transactions`:

```sql
INSERT INTO transactions (user_id, amount, type, description)
VALUES (123456789, -79900, 'purchase', 'Покупка тарифа: 3 месяца (3 мес.)')
```

---

## 🧪 Тестирование

### Ручное тестирование

1. Пополни баланс через админ-панель
2. Открой раздел "Купить / Продлить"
3. Выбери тариф
4. Подтверди покупку
5. Проверь получение ключа

### Проверка через БД

```sql
-- Посмотреть транзакции
SELECT * FROM transactions WHERE type = 'purchase' ORDER BY created_at DESC;

-- Посмотреть пользователей с подпиской
SELECT user_id, expires_at, current_key FROM users WHERE expires_at > datetime('now');

-- Посмотреть баланс
SELECT user_id, balance FROM users WHERE user_id = 123456789;
```
